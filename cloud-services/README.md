# cloud-services
==========================

Microservices projects using Axon Framework and spring cloud basic APIs.

Used APIs,

- Java 8
- Spring Boot 2.2.2
- JPA, Hibernate 5.0
- H2database
- Spring Boot Config
- Eureka
- Zuul
- Axon 
- Feign Client
- RabbitMQ
- RestTemeplate
- Hystrix

Project Structure

- [x] Axon Microservices
- [x] Client Microservices
- [x] Bill Microservices

> This project is in the development stage
